# ANSIBLE ROS2

## Usage
Installing ROS2
```bash
ansible-pull -U https://github.com/brucechanjianle/ansible-ros -e "ros2_distribution=noetic" -K
```

For Docker
```bash
ansible-pull -U https://github.com/brucechanjianle/ansible-ros -e "ros2_distribution=noetic"
```
